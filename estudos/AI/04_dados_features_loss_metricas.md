# Modulo IA 04: Dados, features, loss e metricas

Mesmo num projeto que nao usa backprop, estes conceitos importam porque fazem parte da linguagem geral da area.

## 1. Dados

Todo sistema de ML depende de alguma forma de experiencia estruturada.

## 2. Features

Features sao as entradas escolhidas para representar o estado do problema.

No projeto:

- sensores
- velocidade

## 3. Loss

Loss e uma medida de erro usada tipicamente em treino por gradiente.

## 4. Metrica

Metrica e a forma de avaliar desempenho.

## 5. Fitness vs loss

No projeto atual, o papel central e desempenhado por fitness, nao por uma loss diferenciavel tradicional.

## Exercicios

### Exercicio 1

Explique a diferenca entre feature, loss, metrica e fitness.

### Exercicio 2

Descreva quais sao as features do projeto atual e por que elas fazem sentido.
