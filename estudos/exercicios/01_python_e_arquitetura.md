# Exercicios 01: Python e arquitetura

## Objetivo

Consolidar o jeito Python de estruturar um projeto como este.

## Lista

1. Explique a diferenca entre estado e comportamento usando a entidade `Carro` como exemplo.
2. Crie uma classe `Carro` com `x`, `y`, `angulo`, `velocidade` e um metodo `mover()`.
3. Crie uma classe `Rede` com atributos `W1`, `b1`, `W2`, `b2` e um metodo `copiar_de()`.
4. Escreva uma estrutura de pastas minima para um projeto de simulacao com `main.py`, `config.json` e pasta `sim/`.
5. Carregue um `config.json` em Python e leia pelo menos 3 hiperparametros.
6. Explique por que separar motor de simulacao e visualizacao e uma boa decisao.

## Desafio extra

Implemente um mini-projeto com `main.py`, `carro.py`, `config.json` e `simulador.py`, mesmo que ainda sem IA.
