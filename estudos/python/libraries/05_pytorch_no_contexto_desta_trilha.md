# Modulo Bibliotecas 05: PyTorch no contexto desta trilha

Mesmo que o projeto atual nao use PyTorch em runtime, voce pediu uma formacao que te deixe capaz de desenvolver redes neurais de forma mais ampla. Por isso este modulo existe.

## 1. O que e PyTorch

PyTorch e uma biblioteca de deep learning muito usada para construir e treinar redes neurais com autodiferenciacao.

## 2. O que ele te da que NumPy puro nao te da facilmente

- gradientes automaticos
- otimizadores prontos
- layers prontas
- suporte forte a GPU

## 3. Quando NumPy basta

- estudo de fundamentos
- redes pequenas
- implementacoes didaticas
- neuroevolucao simples

## 4. Quando PyTorch passa a valer muito

- treino supervisionado com backprop
- CNNs, RNNs e Transformers reais
- datasets maiores

## 5. Valor pedagogico

Aprender primeiro com NumPy deixa PyTorch menos magico depois.
