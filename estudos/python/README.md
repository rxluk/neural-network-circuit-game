# Trilha Python

Esta trilha nao e sobre logica de programacao do zero. Ela e sobre aprender o jeito Python de construir este tipo de sistema.

## O que voce vai aprender aqui

- como Python organiza codigo em modulos e classes
- como Python lida com arquivos, JSON e configuracao
- como escrever codigo idiomatico em vez de codigo apenas "funcional"
- como debugar e estruturar um projeto de simulacao
- como gerenciar ambiente, dependencias e execucao
- como evoluir qualidade do codigo sem se perder

## Resultado esperado

Ao fim desta trilha, voce deve conseguir olhar para o codigo do projeto e dizer nao apenas "o que ele faz", mas tambem "por que foi escrito assim em Python".

Depois da expansao desta trilha, voce tambem deve conseguir:

- tratar erros com `try/except` sem esconder bugs
- traduzir rapidamente conceitos Java/PHP para Python idiomatico
- usar slicing, generators, `map/filter/reduce` com criterio

## Exercicios

### Exercicio 1

Antes de ler os modulos, escreva o que voce acha que significa "codigo Python idiomatico". Depois volte e compare.

### Exercicio 2

Abra o projeto e tente identificar, sem ajuda, onde estao as classes principais, os modulos principais e os arquivos de configuracao.
